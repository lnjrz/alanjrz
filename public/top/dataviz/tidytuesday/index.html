Data Visualization & Computational Ecology

A selection of data visualizations I have produced as contributions for the weekly [#TidyTuesday challenge](#TidyTuesday using the {ggplot2} package in the programming language R. All visualizations are the result of the code without any post-processing.

You need a visualization? Let’s get in touch!

#TidyTuesday is a weekly data project aimed at the R ecosystem and the {tidyverse} package bundle to wrangle, explore and visualize the data set that is provided by the R4DS Online Learning Community.
